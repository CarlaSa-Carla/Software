#  SOFTWARE DE GERENCIAMENTO E DIVULGAÇÃO DE MODELOS (E-COMMERCE).

ESCOPO

Objetivos e finalidades: 


O objetivo principal do sistema será representar e promover modelos, otimizando o gerenciamento das agências, permitindo assim o controle eficiente das agências e dos modelos.


Objetivo geral

Desenvolver um software para gerenciamento e divulgação de modelos (e-commerce).


Ojetivos específicos
           
•	Desenvolver um chatbots interativo para facilitar a comunicação eficiente com o cliente.

•	Construir um e-commerce, que serve de suporte para as agências e com suporte multilíngue.

•	Implementar um banco de dados para armazenar as agências do mercado.

•	Manter o controle das agências e modelos do mercado proporcionando segurança ao cliente.

•	Criar uma interface de usuário intuitiva e responsiva que possibilite os clientes, empresas e outros a interagirem de forma simples e eficaz com o sistema.

•	Contribuir no desenvolvimento da indústria de moda, com a criação de um software de gerenciamento e divulgação de modelos.

                                                                               JUSTIFICATIVA DO PROJETO

Este projeto busca prosseguir com uma pesquisa já iniciada e ampliar o estudo, propondo a criação de um software de gerenciamento e divulgação de modelos (e-commerce). Sabe-se que existem agências físicas e também digitais. Mas não existe um plataforma digital que tem como objetivo, gerenciar a agências do mercado da moda, assegurando a veracidade das agências, modelos e a segurança do cliente. Existem muitas pessoas, que tiveram o seu sonho de ser modelo frustrados, devido a inúmeras agências falsas formadas com intuito de roubar dinheiro dos clientes aproveitando-se de seu sonho. Ainda devido a inúmeras agências, é difícil saber ao certo ou levaria muito tempo para verificar se um modelo que não é famoso existe ou se foi afastado da agência. Visto que a indústria da moda é dinâmica e sempre está à procura de novos talentos, logo surge a necessidade da criação de um (e–ccomerce) que será como se fosse uma agência mãe do mercado da moda.
